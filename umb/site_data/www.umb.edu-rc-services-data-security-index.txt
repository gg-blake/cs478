We can help review data security plans necessary to ensure compliance  with intellectual property requirements, data privacy regulations and/or  contractual obligations such as Data Use Agreements.  Data Use  Agreements are negotiated through the Office of Research and Sponsored  Programs, but we can assist with structuring the associated data security plan and with structuring the local computing environment to  ensure compliance.
